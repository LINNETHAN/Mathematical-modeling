# TOPSIS法代码
（暂不包含权重，以清风课程例题为例）
TOPSIS法,又称优劣解距离法
评价类类型的一种模型算法，可结合层次分析法添加权重计算
## 层次分析法的局限性
1. 决策层不能太多，太多则n会很大，判断矩阵和一致性矩阵的差距就会很大
2. 决策层指标通常需要自己确定，当给定指标时，需要利用给定数据使评价更加准确
## TOPSIS法步骤
1. 数据处理
2. 原始矩阵正向化
3. 正向化矩阵标准化
4. 计算得分并归一化
